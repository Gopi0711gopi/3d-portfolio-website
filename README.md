# 3D Interactive Portfolio Website

> An immersive 3D portfolio website built with Next.js 14, Three.js, and React Three Fiber.

## Overview

A visually stunning personal portfolio featuring interactive 3D scenes, smooth scroll animations, and GPU-accelerated rendering — built to stand out with WebGL-powered visuals.

## Features

- 🌐 **3D Scenes** — Interactive Three.js models rendered with React Three Fiber
- 🎬 **Smooth Animations** — Framer Motion page transitions and scroll-linked effects
- 🎨 **Spline Integration** — Real-time 3D model interaction via Spline runtime
- ⚡ **Next.js 14** — App Router, SSR, and optimized image loading
- 📱 **Responsive** — Mobile-friendly layout with Tailwind CSS
- 🌙 **Dark Theme** — Premium dark aesthetic throughout

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **3D:** Three.js, @react-three/fiber, @react-three/drei
- **Spline:** @splinetool/react-spline, @splinetool/runtime
- **Animation:** Framer Motion
- **Styling:** Tailwind CSS
- **Language:** TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

```bash
npm run build   # Production build
npm start       # Start production server
```
